USB 3.0(및 그 이후 규격) 메모리의 속도는 크게 **이론적인 인터페이스 속도**와 **실제 메모리 칩의 쓰기/읽기 속도**로 나뉩니다.

### 1. 규격별 이론상 최대 속도 (인터페이스 속도)
USB 명칭이 여러 번 바뀌어 혼동될 수 있으나, 현재 통용되는 기준은 다음과 같습니다.

| 공식 명칭 (USB 3.2 기준) | 과거 명칭 | 이론적 최대 속도 | 특징 |
| :--- | :--- | :--- | :--- |
| **USB 3.2 Gen 1** | USB 3.0 / 3.1 Gen 1 | **5Gbps** (약 625MB/s) | 가장 일반적인 파란색 단자 |
| **USB 3.2 Gen 2** | USB 3.1 Gen 2 | **10Gbps** (약 1,250MB/s) | 주로 빨간색 또는 USB-C 단자 |
| **USB 3.2 Gen 2x2** | - | **20Gbps** (약 2,500MB/s) | 최신 고성능 외장 SSD급 |
| **USB4** | - | **40Gbps ~ 80Gbps** | 썬더볼트와 호환되는 최상급 규격 |



### 2. 실제 체감 속도 (실제 사용 시)
위의 수치는 '길(인터페이스)'의 넓이일 뿐, 실제로 데이터를 주고받는 '차(메모리 칩)'의 속도는 제품 등급에 따라 천차만별입니다.

* **보급형 USB 3.0 메모리:**
    * **읽기:** 70~150MB/s
    * **쓰기:** 10~40MB/s (USB 2.0보다 약간 빠른 수준인 경우가 많음)
* **고성능 USB 3.1/3.2 메모리:**
    * **읽기:** 300~450MB/s
    * **쓰기:** 100~300MB/s (대용량 영상 파일 전송 시 유리)
* **외장 SSD형 메모리:**
    * **읽기/쓰기:** 500~1,000MB/s 이상 (SATA/NVMe 기술 적용 제품)

### 3. 속도에 영향을 주는 요인
1.  **메모리 타입:** MLC/TLC 등 칩의 종류에 따라 수명과 쓰기 속도가 결정됩니다. 저가형은 데이터를 조금만 옮겨도 속도가 급격히 떨어지는 현상이 발생합니다.
2.  **파일 크기와 개수:** 10GB 파일 하나를 옮기는 것이 1MB 파일 10,000개를 옮기는 것보다 훨씬 빠릅니다. (자잘한 파일은 오버헤드 때문에 속도가 급감함)
3.  **포트 호환성:** 컴퓨터 단자가 USB 2.0(검은색)이면 아무리 빠른 USB 3.2 메모리를 꽂아도 30~40MB/s로 제한됩니다.

**결론:** 단순 문서 보관용이라면 일반 **USB 3.0(Gen 1)**으로 충분하지만, 고해상도 영상이나 대용량 데이터를 자주 옮긴다면 **USB 3.2 Gen 2** 이상을 지원하고 '쓰기 속도'가 명시된 제품을 선택하는 것이 좋습니다.
